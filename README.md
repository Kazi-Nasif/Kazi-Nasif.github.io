# Kazi Fahim Ahmad Nasif Portfolio

Static portfolio website for Kazi Fahim Ahmad Nasif.

## Local Preview

Open `index.html` in a browser, or serve the directory:

```bash
python3 -m http.server 8000
```

## GitHub Pages

This site is ready for GitHub Pages. For a user site, publish it to:

```text
Kazi-Nasif/Kazi-Nasif.github.io
```

GitHub Pages will serve it at:

```text
https://Kazi-Nasif.github.io
```
